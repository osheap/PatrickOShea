Notable Meteor Impacts

This app displays notable meteor impact craters from the University of New Brunswick's Earth Impact Database. Users can explore notable impact craters and glean useful information such as the age of the crater and its diameter.

Historic Hurricane Tracks Web Application

This app displays historic hurricane tracts from the national weather service and color-codes them based on category. Users can use the slider to display tracts from different years, and click on the interactive timestamps to follow each of the storms as they progress.

Historic Kilauea Lava Flows

This application displays historic lava flows from the Kilauea Volcano, between 1791 and 1982. The original GIS data was obtained from the USGS and can be found by searching doi:10.5066/P9B7904I. The slider allows users to cycle between the years 1791 and 1982, which updates the lava flows displayed and total lava flow area counter.

Water Body Generator

Repository currently contains a QGIS Plugin I created using python that generates waterbody features from a json input. The tool outputs a geopackage of polygon and polyline waterbody features. The files include scripts for the tool's backend functionality, custom class definitions, the GUI's user interface, and an example json file that can to test the tool. The readme file within teh waterbodyjsongenerator folder details installation instructions for the GUI.
